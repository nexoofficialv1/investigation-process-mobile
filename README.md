# Investigation & Process Mobile MVP v1.5

Mobile-first police investigation drafting app.

## v1.5 changes
- Strict official CD format correction from user sample.
- CD-I includes PO, DO, DR, DD, DA, RO, IO opening details.
- CD entry table no longer draws horizontal lines between individual diary entries.
- Continuation CD page header support.
- Sketch map symbols look like house/shop/pond/tree/road/field instead of generic circles.
- 35(3), 94 BNSS and Forwarding templates updated toward the supplied official style.

## Build
Push to GitHub and run the Android APK workflow.

### v1.6 update
- PDF and DOC export are both available from Preview.
- CD data entry is now entry-wise and mobile-scroll friendly: Entry No/Time, Place of Entry, Synopsis, and Proceedings are separate editable fields.
